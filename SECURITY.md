# Security

This repository is a compatibility pointer for older UnisonOS specifications and does not operate a standalone network service.

For current security guidance and canonical specifications, see:

- https://project-unisonos.github.io/reference/security/
- https://github.com/project-unisonOS/unison-docs

If you believe you have found a security issue in the documented specifications or migration guidance, please follow the responsible disclosure guidance in the UnisonOS security docs or open a private security advisory on GitHub instead of filing a public issue.
